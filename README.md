# GudWin BookS / ГудВин Букс

Полностью новый визуальный проект с нуля:
- `index.html` - главная страница выбора (`Сказки` или `Комиксы`)
- `fairy.html` - отдельный раздел сказок
- `comics.html` - отдельный раздел комиксов
- `assets/css/main.css` - яркий стиль + анимации
- `assets/js/*.js` - логика генерации, анимации, tabs, particles
- `legal/*.html` - юридический блок РФ 2026

## Локальный запуск

```bash
cd "/Users/macbook/Documents/New project/gudwin-books-fresh"
python3 -m http.server 4180
```

Открыть: `http://127.0.0.1:4180`

## Примечание

Это frontend MVP с демонстрационной генерацией. Реальная Telegram OAuth, Telegram Stars billing, AI text/image/TTS и backend очередь подключаются на следующем этапе.
